## Setup
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn src.main:app --reload
```

- ReDoc: http://localhost:8000/redoc

```bash
cd backend
python3 -m venv venv
source venv/bin/activate

```
